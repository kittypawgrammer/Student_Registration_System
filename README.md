# 🎓 Student Registration System

## 📌 Project Description

The **Student Registration System** is a responsive web-based application developed using **HTML, Tailwind CSS, and JavaScript**.
This system allows users to register student details, manage records, edit existing entries, and delete records easily.

All student data is stored in **Local Storage**, ensuring that the data remains available even after refreshing the page.
The system is designed with a clean and modern user interface and is fully responsive across **mobile, tablet, and desktop devices**.

---

## 🎯 Objective

The main objective of this project is to build a student registration system that:

* Registers student details
* Displays student records
* Allows editing and deleting of records
* Validates input fields
* Stores data using local storage
* Maintains a clean and responsive design
* Uses proper Git commit structure

## 🚀 How to Run the Project

### Step 1: Download or Clone the Project

git clone https://github.com/kittypawgrammer/Student_Registration_System.git

or download zip file and extract.

---

### Step 2: Open Project Folder

cd student-registration-system

---

### Step 3: Install Dependencies

npm install

---

### Step 4: Run Tailwind

npx tailwindcss -i ./src/input.css -o ./dist/output.css --watch

---

### Step 5: Open in Browser

Open **index.html** in your browser.

---

## 📝 Form Validation Rules

* Student Name → Only characters allowed
* Student ID → Numbers only
* Email → Valid email format
* Contact Number → Minimum 10 digits
* Empty fields are not allowed

---

## 💾 Data Storage

* Student data is stored in **Local Storage**
* Data will not disappear after page refresh
* Records can be edited and deleted anytime

---

## 📱 Responsive Design
